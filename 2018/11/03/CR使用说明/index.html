Cat-R使用说明

  1. cat-radar本身就提供游戏加速，==使用之前先关闭其他加速器==，否则会有冲突。
  2. 苹果与其他平台，都请使用==谷歌浏览器==。
  3. 游戏电脑需要安装SSTap，请先点此下载吃鸡专用版。
  4. 雷达不用的时候在页面上点击关闭，不然会消耗剩余时间。
  5. 如有其他疑问，请联系群主。

网页登录入口

1. 解压下载下来的SSTap，并运行==RuleUpdate.exe==；

2. 点击Cat-R后台的启动按钮，并选择一个您想使用的加速节点地区，点OK后开启加速服务（==国内推荐“香港A/B”==，大约需要1分钟左右）；

3. 等待服务器开启的过程中，可以先在SSTap界面中选择模式为“绝地求生(最新版本号)-”，并点击“+”–“添加一个SS/SSR代理”；

4. 待服务器启动完成后，将服务器信息填入到SSTap对应的文本框中，并点击下方保存按钮；

5. 在SSTap界面上选择刚添加的代理，并点击右侧的闪电按钮进行测试，TCP与UDP测试均为绿色字为正常，这时点击下方的“连接”按钮，连接成功后SSTap界面会自动最小化；

6. 此时您可以启动游戏了，如果在此之前您已经启动了游戏，需要重新运行游戏；

7. 点击Cat-R后台的观看按钮，选择您喜欢的方式观看雷达~
